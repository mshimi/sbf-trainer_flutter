SBF Trainer is a specialized cross-platform mobile application designed to assist users in preparing for the German Sport Boat License (SBF Binnen). The app provides an efficient, location-independent learning environment by integrating the complete official ELWIS question catalogue.

Key Features:

Official Content: Includes all 300 official questions and nautical images.

Offline Availability: Fully functional without an internet connection using local SQLite storage.

Smart Tracking: Visualizes learning progress and mastery statistics via a dedicated dashboard.

Instant Feedback: Provides immediate visual feedback on answers during training sessions.
